Introduction to Python

What is Python?

Python is a high-level, interpreted programming language known for its simplicity and readability. It was created by Guido van Rossum and first released in 1991. Python's syntax emphasizes code readability, making it an excellent choice for beginners and experienced programmers alike is versatile and widely used across various fields, including web development, data analysis, artificial intelligence, scientific computing, and automation. It supports multiple programming paradigms, including procedural, object-oriented, and functional programming.

Python Platforms / Editions

There are platforms or editions of python:

    Python is a versatile programming language that can run on various platforms, including Windows, macOS, Linux, and Unix. Python itself is not divided into different "editions" in the same way that some software platforms are. However, there are different versions of Python, with the two main branches being Python 2 and Python 3. As of January 1, 2020, Python 2 has reached its end-of-life, and Python 3 is the actively developed and maintained version. Python 3 is further divided into releases, with the most recent stable release being Python 3.10.x as of my last update in January 2022. Each release brings new features, improvements, and bug fixes. Python can be installed on your system directly from the official Python website (https://www.python.org/), or through package managers like pip or Conda. History of python

Python was created by Guido van Rossum, and first released on February 20, 1991. While you may know the python as a large snake, the name of the Python programming language comes from an old BBC television comedy sketch series called Monty Python’s Flying Circus. One of the amazing features of Python is the fact that it is actually one person’s work. Usually, new programming languages are developed and published by large companies employing lots of professionals, and due to copyright rules, it is very hard to name any of the people involved in the project. Python is an exception. Of course, Guido van Rossum did not develop and evolve all the Python components himself. The speed with which Python has spread around the world is a result of the continuous work of thousands (very often anonymous) programmers, testers, users (many of them aren’t IT specialists) and enthusiasts, but it must be said that the very first idea (the seed from which Python sprouted) came to one head – Guido’s.

Python Variables

Listed python variables

    Variables in Python are containers for storing data values. They can hold different types of data, including numbers, strings, lists, dictionaries, etc. Python variables are dynamically typed, meaning you don't need to declare their type explicitly. Variable names are case-sensitive and must start with a letter or underscore. You can assign values to variables using the assignment operator (=). Variables can be reassigned to different values during program execution. Python supports multiple assignment, allowing you to assign multiple variables in a single line. Variables can be used in expressions and statements to manipulate data. The del keyword can be used to delete variables and free up memory. Variables exist within a scope, and their visibility is determined by the block in which they are defined. Python follows the LEGB (Local, Enclosing, Global, Built-in) scope resolution rule to determine the value of a variable. Variable naming conventions often follow the snake_case or CamelCase conventions for readability. It's a best practice to use descriptive names for variables to make code more understandable.
